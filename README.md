# tfm
A light weight TUI file manager made in C
## Features
- Cross platform
    - [ ] Support for Windows
    - [ ] Support for MacOS
    - [x] Support for Linux
- Easy to edit config files
- Subprocess shell
- Keybinds
    - [x] `↑`: (Up arrow key) Select above
    - [x] `↓`: (Down arrow key) Select below
    - [ ] `→`: (Right arrow key) Open file or open directory
    - [ ] `←`: (Left arrow key) Go to parent directory
    - [ ] `c`: Cut file
    - [ ] `y`: Copy file
    - [ ] `p`: Paste file
    - [ ] `m`: Move file
    - [ ] `d`: Delete file
    - [ ] `u`: Undo
    - [ ] `l`: Change directory
    - [ ] `h`: Display keybinds
