# tfm
A light weight TUI file manager made in C
## Features
- Cross platform
    - [ ] Support for Windows
    - [ ] Support for MacOS
    - [x] Support for Linux
- Easy to edit config files
- Subprocess shell
- Keybinds
    - [ ] `o`: Open file or open directory
    - [ ] `c`: Cut file
    - [ ] `y`: Copy file
    - [ ] `p`: Paste file
    - [ ] `d`: Delete file
    - [ ] `u`: Undo
    - [ ] `l`: Change directory
    - [ ] `h`: Display keybinds
